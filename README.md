# 鐵人賽 Day 21：Rspack / Rsbuild 怎麼用

[文章連結](https://ithelp.ithome.com.tw/users/20169399/ironman/8023)

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
